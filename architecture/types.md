## 型
### 基本型

- Int
	- C++ の std::int64_t 型と対応
- Bool
	- C++ の bool 型と対応
- Float
	- C++ の double 型と対応
- Text
	- 文字列 （2^32 バイト未満）
- Video
	- 映像型
	- height, width は 2^32 未満
	- フレーム数は 2^64 未満
		- 将来的に生放送に対応した場合に桁あふれを防ぐために多めに桁数を持たせている
- Audio
	- 音声型
	- 詳細は未定
- Vector 上記の型の可変長配列 （2^32 個未満）
	- 中に Vector は入れられない
